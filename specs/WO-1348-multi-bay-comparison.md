# WO-1348 — Multi-Bay Comparative Analysis

**STATUS:** BACKLOG  
**DATE:** 2026-05-30  
**DEPENDS ON:** WO-1347

## Purpose

Six independently loaded bays viewed simultaneously. KRYLO becomes a workstation.

## Example State
```
Cone 01 = NVIDIA
Cone 02 = PALANTIR
Cone 03 = ANDURIL
Cone 04 = OPENAI
Cone 05 = EMPTY
Cone 06 = EMPTY
```

## Visual
Each loaded cone = an intelligence cartridge. Label visible on cone. Stats panel switches context when cone is clicked.

## Comparison Mode
When two or more bays are flagged for COMPARE (WO-1347), a cross-bay analysis view activates showing signal deltas, convergence alignment, and divergence vectors between the loaded subjects.
